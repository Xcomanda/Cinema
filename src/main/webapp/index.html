Cinema trainee project

  • Посмотреть расписание
    http://host:port/Cinema/webapi/schedule
    Метод GET
  • Посмотреть бронь по id
    http://host:port/Cinema/webapi/order/{id}
    id - уникальный номер брони
    Метод Метод GET
  • Создать бронь
    http://host:port/Cinema/webapi/order?{movieId}=[number]&{tickets}=[number]&{date}=[date]
    movieId - уникальный номер фильма
    tickets - нужное количество билетов
    date - дата (формат ввода задается в файле свойств)
    Бронь не будет создана если указан несществующий id или дата показа
    Метод POST
  • Удалить бронь по id
    http://host:port/Cinema/webapi/order/{id}
    id - уникальный номер брони
    Метод DELETE

Файл свойств properties.properties

  • storage_path - где хранятся файлы расписания и заказов
  • storage_date_format - формат даты для хранения
  • query_date_format - формат даты для запросов
  • schedule_file_name - имя файла расписания
  • orders_file_name - имя файла заказов
